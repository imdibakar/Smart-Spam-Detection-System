# Smart-Spam-Detection-System
This project presents a Smart Spam Detection System that classifies messages into spam or ham(normal) using a ruling Artificial Intelligence approach. 
---------------------------------------------------------------

## 📌 Features
--------------------------------------------
1. It detects spam messages using predefined keywords
2. It is simple and fast execution
3. It allows user to add new words which is it's leaning feature.
-----------------------------------------------------------------

## 📌 Objective
--------------------------------------------
1.  To design a system that can detect spam messages
2.	To classify messages as spam or ham i.e., normal
3.	To implement a simple AI-based solution using python
4.  To allow the system to learn new words dynamically
-------------------------------------------------------------------

## 🧠 How it works
-----------------------------------------------
1. The system is trained using spam words such as win, free, money, lottery, etc., and ham words such as hello, tomorrow, meeting, project, etc.
2. Users enter a message
3. The message is converted to lowercase using .lower() and split into words
4. The system checks that if any word matches with spam-"Spam" or else if it matches with ham- "Ham"
